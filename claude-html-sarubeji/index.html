Claude HTML サルベージ

Claudeの全トーク履歴(エクスポートデータ)からHTML・Reactアプリを掘り出して、最新のものから順に並べます。検索して、プレビューして、そのままダウンロード。

エクスポートデータの入手方法
  1. claude.ai の 設定 → プライバシー → データをエクスポート をクリック
  2. 届いたメールからデータ(zip)をダウンロード
  3. その zipファイルごと、または中の conversations.json を下にドロップ
