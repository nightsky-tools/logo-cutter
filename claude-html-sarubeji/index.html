Claude HTML サルベージ

Claudeの全トーク履歴(エクスポートデータ)からHTML・Reactアプリを掘り出して、検索・プレビュー・ダウンロード。気に入ったものはライブラリに保存して、2台の端末で同期・バックアップできます。

エクスポートデータの入手方法
  1. claude.ai の 設定 → プライバシー → データをエクスポート をクリック
  2. 届いたメールからデータ(zip)をダウンロード
  3. その zipファイルごと、または中の conversations.json を下にドロップ

読み込み中…(大きいファイルは少し時間がかかります)

ファイルの代わりにテキストを貼り付ける

JSONを含むテキストや、コードブロック入りのメッセージをそのまま貼り付けても抽出できます(上限 20MB)。
